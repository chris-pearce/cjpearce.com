# My personal website

<http://cjpearce.com>.

## Install

Check if Jekyll is installed:

```bash
jekyll --version
```

If it isn't run:

```bash
gem install jekyll
```

To install all website dependencies:

```bash
npm install
```

## Dev

```bash
npm start
```

If it doesn't run due to Jekyll issue run:

```bash
bundle update
```

## Build

```bash
npm run build
```
